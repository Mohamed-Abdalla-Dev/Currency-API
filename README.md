# 💱 Currency Converter

A simple currency converter built with JavaScript.  
It fetches real-time exchange rates from an API and converts between currencies.

## 🚀 Features

- Convert between different currencies  
- Fetch real-time exchange rates  
- Simple and clean UI  

## 🛠 Technologies Used

- HTML  
- CSS  
- JavaScript(Vanilla)  
- API (Fetch)  

## 📌 Goal

Practice working with APIs, fetch requests, and handling real-time data.
